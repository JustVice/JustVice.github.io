Program description

NodeJS webserver which can be used to send emails sending a HTTP POST request. SMPT service provided by SendGrid.

Live demo

This software is used to power up the Send a Message system designed for this page.

Program features
  • Private email and almost unlimited sending.
  • Relatively easy to configure.
  • Ultra fast and easy to configure.
  • Secure (SendGrid service).
