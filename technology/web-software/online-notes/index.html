Program description
  • Working site: Online Notes.

Publish notes on the web publicly or privately and check them again by coming back to the web page.

The idea behind Online Notes is to be able to exchange quick note strings between devices.

For example: passing an URL you have on your computer to your device by publishing the URL into the page, visiting the page again at your mobile device and to copy the URL back again.

No demo images yet.
Program features
  • Store private notes by making an account.
  • Secure passwords storage.
  • API available.
  • Very fast performance.
  • Simple and straight to the point user interface.
  • Responsive design (can be used on mobile devices).
