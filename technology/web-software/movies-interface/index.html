Program description

Movies Interface is a tiny software for web servers which allow you to stream videos stored in your computer on the local network.

Movies Interface needs a web server host software to run. It has been developed as running on Apache2. Therefore Apache2 software should be used to ensure correct functionality.

I recommend installing XAMPP which is a software that gives a minimal user interface to manipulate Apache2 service.

Note
Movies Interface uses HTML5 web player which means only the following formats are supported:

  • mkv
  • mp4
  • WebM
  • ogg
  • ogg
  • mov

Please, go to Movies Interface's Github repository to read the guidelines of how to setup Movies Interface.

Program features
  • Easy to configure.
  • Feather size.
  • Light local network bandwidth consumption.
  • Compatible with most browsers.
  • Responsive to screen size (can be used on mobile devices).
  • Link to direct download the movies integrated.
