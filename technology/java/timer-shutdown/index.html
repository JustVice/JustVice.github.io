Timer Shutdown

Program specifications
  • Author: VICE
  • License: MIT
  • Latest version: 1.5
  • Programming language: Java
  • Language: English
  • First release date: 8 October 2018.
  • Compatibility: Only Windows.
Program description

Sets an inner timer inside your Windows operating system to shut down by itself at a given time.

Program features
  • Set a timer to shutdown your PC automatically by itself.
  • Cancel the shutdown timer by clicking the cancell button.
  • Extremely easy to use.
