Program description

A simple Java desktop application to create dossiers.



Note: Dossiers are stored in a file called "db.db". Editing .txt generated files in "Dossiers files" folder won't take effect on database stored files. These files are for read-only purposes. Any change in the content of this folder will be replaced with "db.db" content. Editing content inside "Dossiers files" folder will result in data lost next time application is opened.
Program features
  • Locally secure (no data is sent through the Internet)
  • Dossiers get stored as .txt files.
  • Create, read, update, and delete dossiers.
  • Works on any platform with Java installed.
  • Out of the box use.
  • Simple and fast.
