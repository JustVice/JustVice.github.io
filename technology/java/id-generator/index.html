Program description

Generates random IDs from different lengths for you by just clicking a button.

Program features
  • Possibility to create manual IDs with the name/pattern you wish.
  • Copies IDs automatically to clipboard and by pressing a button.
  • Possibility to create prefixes and add them to the ID.
  • Option to generate only numeric IDs.
