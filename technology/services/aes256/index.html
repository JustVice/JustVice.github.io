AES256 Service
Encrypt and decrypt text
Encrypt text strings with a password key given by you. Come again anytime to decrypt your encrypted text with the same password key given.

About
  • All information given to encrypt is not stored.
  • Information is send through secure HTTPS channel.


Troubleshooting

If the decoded result is not readable, it means password key is not valid.


AES service locally
Still feeling distrustful? You can try my local application: V-AESEcrypt-2.
Note: Encrypted text with online version and V-AESEcrypt-2 are not compatible. Text encrypted with either online or local version can only be decrypted with same software.
