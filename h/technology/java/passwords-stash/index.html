Passwords Stash

Program specifications
  • Author: VICE.
  • Latest version: 2.3
  • Programming Language: Java.
  • Language: English.
  • First release date: 1 June 2018.
Program description

A storage/stash for your passwords. Designed to give your passwords as fast as it can by making double click over the list of stored passwords.
Equipped with light security protocols to keep the data secure.
You can use it to store all your passwords, however, its idea is to just be a quick passwords giver.

Notes from autor: I have a special love for this program due to its initial difficulty and the challenges it brought to me when it was a baby.

Program features
  • Save, edit, and delete passwords info as you wish
  • Optional Master Password system to keep access limited
  • Delete data after failing the master password in a settled number of times system
  • Passwords information codified
  • Languages: English.
  • Intuitive
  • Double click to select copy to clipboard passwords
  • Additional notes for the passwords
  • Favorite set system
  • Open source
  • Function to print all the passwords inside .txt files
