Program description

A lite version of Passwords Stash made to run on any Windows PC without the necessity of installing Java.

This program is intended for people who usually log in to their accounts in different PC's all the time to do their work. Passwords Stash and Passwords Stash Lite give a quick solution to log in faster by just promoting a point-and-click on a list. Carrying it on a USB Pendrive.

How it works

Open your list of passwords you ateriorely stored and doble click any item inside the list to copy it to your clipboard and to paste in inside any password box to quicly login.

Discarted features:
  • Printing passwords.
  • Encrypted storaged data.
  • To edit Master password (you must delete current to change it).
  • Attempts to try Master Password and delete all data
  • Passwords trash can.
  • To mark passwords as favorite.
  • Search engine to search for passwords by service name.
  • Intuitiveness was reduced, but still easy to use.
