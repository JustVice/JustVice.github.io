Program description

A lite version of Passwords Stash made to run on any Windows PC without the necessity of installing Java or other software.

This program is intended for people who usually log in to their accounts in different PC's all the time to do their work. Passwords Stash and Passwords Stash Lite give a quick solution to log in faster by just promoting a point-and-click on a list. Carrying it on a USB Pendrive.

How to use

  • Save a credential information and come back later.
  • Open the program and go to your passwords stored list.
  • Double click any service listed to copy the password on clipboard.
  • Paste your password inside the password text box on your browser.
  • Done!

Security

  • To encrypt the data stored enable a Master password.
  • The Master Password will be a key to encrypt all the data with AES encryption.
  • If you lose your Master Password, it would not be possible to restore the data encrypted.
  • If you are planning to use Passwords Stash Lite to store all your passwords, I recommend using a password manager such as LastPass.
  • Passwords Stash Lite is intended to use how it was stated at the how to use section.

Discarted features:
  • Printing passwords.
  • To edit Master password (you must delete current to change it).
  • Attempts to try Master Password and delete all data.
  • Passwords trash can.
  • To mark passwords as favorite.
  • Search engine to search for passwords by service name.
  • Intuitiveness was reduced, but it is still easy to use.
